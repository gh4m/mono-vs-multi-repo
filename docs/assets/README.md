# Documentation Assets

Use this directory for diagrams, screenshots, and other visual aids referenced from the docs.

Guidelines:

- Use kebab-case filenames (for example, `monorepo-ci-flow.png`).
- Prefer vector formats (`.svg`) when possible; fall back to `.png` for raster images.
- Keep source files for generated diagrams here as well (for example, `.drawio`, `.plantuml`).
